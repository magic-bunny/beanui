# BeanUI


The JAVA Web framework

## Dependencies
1. install nodejs or npm;
2. install apache maven;

## Getting Started
1. `cd beanui/vue`;
2. `npm install`;
3. `cd beanui`
4. `mvn install`
5. `cd beanui/demo`
6. `java -jar demo-[version].jar`
